## Usage rule

When you point a user to docs:

- link to the narrowest relevant page, not only the docs home
- prefer `better-drizzle.com` URLs over raw repo file paths when the user needs conceptual guidance
- prefer repo file paths when the user needs implementation truth from the current checkout
